# frontend-vite basic app example

Runnable example for the Reikujo `frontend-vite` variant.

This app demonstrates:

- source-owned UI primitives
- `tailwind-variants` primitives
- `Label` + `Input` + `Textarea` form rhythm
- `Badge`, `Separator`, and `Skeleton`
- module-level query-state rendering
- service/type ownership
- hook section comments

## Run

```bash
pnpm install
pnpm dev
```

## Validate

```bash
pnpm typecheck
pnpm test -- --run
pnpm build
```

This example is not a starter template. It is a reference for how generated pieces can be composed.

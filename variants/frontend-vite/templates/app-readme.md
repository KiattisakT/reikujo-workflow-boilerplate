# Frontend App

React + TypeScript + Vite app generated for a Reikujo `frontend-vite` project.

## Commands

```bash
pnpm install
pnpm dev
pnpm typecheck
pnpm lint
pnpm test
pnpm build
```

## Component structure

```text
src/components/ui/
src/lib/utils.ts
```

This app uses source-owned shadcn-style base components. Add packages with exact versions:

```bash
pnpm add -E <package>
pnpm add -D -E <package>
```

Update this file after the real app stack, routing, styling, and test setup are finalized.
